For executing the Matlab coding Assignment Problems,

1. Open the initial.m program and run this file. Both, sum-product and max-product is called from this file.
2. First, enter the number of nodes.
3. Enter the values for the adjacency matrix row-wise.
4. Next, enter the number of iterations. 
5. Then, enter the size for the weight matrix and their values.
6. The solution for max-product and sum-product is displayed.
